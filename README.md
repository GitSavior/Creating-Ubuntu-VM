# Creating-Ubuntu-VM

Download Unbuntu ISO and creating a Ubuntu VM

## Steps

### Step 1: Downlaod Ubuntu ISO

#### Search Ubuntu from google and click the "Download" link

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/67500b54-691a-4c2c-bbe9-aca215ab0e0a)
![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/430573b2-54f7-46b5-a8af-21619b8e788d)

##### Click the "Download Ubuntu Desktop" Button

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/2006edae-e39c-4dda-a42b-0ce4d2b1479a)

#### Click the "Download 22.04.4" Button

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/e32c66a4-1a7a-42e1-88b5-373687d20965)

### Step 2: Creaing an Ubuntu VM

#### Once the download is compelete open VirtualBox and click "New"

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/0f045fed-7c5a-4417-8f93-b76a19182e37)

#### Input a name and select the unbutu-22.04.4-desktop-amd64.iso as the ISO image. Click "Next"

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/1c3dbc62-4276-4e42-90cd-b871aa5ac379)

#### Check the "Guest Additions" Box so that the machine will have copy and paste between its host and itself. Click "Next"

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/6cba3211-db97-4d3c-826a-9feedb9139fb)

#### Select desired Base Memory (ram) and desired amount of processors. Click "Next"

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/5b9a7a2d-8508-4fa1-b7c7-cd95320a28ce)

#### Select enough virtual hard disk for the OS and space desired space. Click "Next"

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/73fa5d93-61c5-41af-89e8-2c3099cab176)

#### Click "Finish" to start setup

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/bffff2dc-300a-4fdb-9564-e3cae1cb9da0)

#### Should see the Ubuntu VM setting up

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/1a19ae69-425a-4c22-86cc-efe1a50f0f67)

#### The Ununtu VM is now complete

![image](https://github.com/GitSavior/Creating-Ubuntu-VM/assets/162067776/b0dadd98-f78c-470e-80f9-eef11c994d89)
